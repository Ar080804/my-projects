### 🏠 [Roomble — Property Discovery & Matching Platform](https://github.com/Shlok-Jain/Roomble)
> **MERN Stack** | *2025*

### 🏢 Roommate Finder REST API & React Frontend
> **Node.js, Express, PostgreSQL, React, Vite** | *2026*
* Built a complete RESTful API using **Express and PostgreSQL** for managing property listings, user profiles, and compatibility scoring.
* Implemented **JWT-based authentication** with secure role-based access control (Tenants vs. Landlords).
* Engineered a sleek, glassmorphic **React frontend** utilizing modern Context API for state management and real-time database filtering.

### 🧪 Project Isotherm: Interactive Ag-Mg Binary Phase Oracle
> **C, C++, Python, SQL** | *May 2026*
* Architected a **web-based Interactive Binary Phase Diagram** tool for the Ag-Mg system, digitizing phase boundaries into piecewise-linear coordinate arrays.
* Engineered a thermodynamic **phase-prediction engine** in C/C++ achieving **±10°C / ±2 at.%** prediction accuracy across the binary phase space.
* Implemented the Lever Rule algorithm to compute equilibrium phase fractions in real-time, coupled with representative microstructure visualizations.

### 🔐 Hamming PUF Cryptanalysis
> **Python, NumPy, scikit-learn** | *June 2026*
* Derived a mathematical feature map that linearizes the response function of a **32-bit Hamming Physical Unclonable Function**.
* Trained a **LinearSVC** model via scikit-learn, systematically tuning hyperparameters to maximize test accuracy.
* Demonstrated that **>99% test accuracy** is achievable with a compact, efficiently computable feature representation.

### 🚙 Bluetooth-Controlled Mars Rover
> **C/C++, Arduino, ESP32** | *2025*
* Designed and fabricated a Mars Rover prototype integrating an **ESP32 microcontroller** with dual DC motor assemblies using differential-drive kinematics.
* Developed embedded **C/C++ firmware** translating smartphone gamepad inputs into motor velocity vectors via trigonometric decomposition.
* Integrated the **DabbleESP32** Bluetooth library for real-time wireless control and rapid maneuverability.
